## ✨ Features

- **Matrix Rain Hero** — Custom HTML5 Canvas animation with cybersecurity aesthetic
- **Typewriter Role Switcher** — Animated role text cycling through key specializations
- **Scroll Reveal Animations** — GSAP-powered entrance animations on scroll
- **Custom Cursor** — Dot + follower cursor with hover effects
- **Project Cards with Images** — Hover overlays with GitHub links
- **Certifications Section** — Education + credentials at a glance
- **Contact Form** — Formspree-powered with async submission
- **Fully Responsive** — Mobile-first, works on all screen sizes
- **Dark Cybersecurity Theme** — Teal/green accent, deep dark background

## 🚀 Deploying on GitHub Pages

1. Push all files to a GitHub repo named `portfolio` (or `<username>.github.io`)
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://<username>.github.io/portfolio`

## 📸 Adding Real Project Screenshots

For best results, replace the Unsplash images in `index.html` with actual screenshots:

```
portfolio/
└── images/
    ├── temp-humidity-project.png
    ├── water-level-project.png
    ├── network-scanner.png
    └── portfolio-screenshot.png
```

Then update the `<img src="...">` in each project card to point to your local images.
